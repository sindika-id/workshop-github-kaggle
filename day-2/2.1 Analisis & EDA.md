## Kenalan dengan Dataset Iris

> **Dataset Iris** = kumpulan data ukuran bunga Iris dari 3 spesies:
> - *Iris setosa*
> - *Iris versicolor*
> - *Iris virginica*

Untuk tiap bunga, kita punya 4 ukuran (dalam cm):
- Panjang sepal (`SepalLengthCm`)
- Lebar sepal (`SepalWidthCm`)
- Panjang petal (`PetalLengthCm`)
- Lebar petal (`PetalWidthCm`)

Total: **150 bunga** â†’ 50 per spesies.

## Langkah-Langkah EDA (Praktik di Kaggle Notebook)

### Langkah 1: Muat Data & Lihat Sekilas

```python
import pandas as pd
df = pd.read_csv('/kaggle/input/iris/Iris.csv')
df.head()
```

![alt text](images/2.1.1-new.png)

**Penjelasan:**
- `pd.read_csv(...)` â†’ memuat file CSV ke dalam bentuk tabel (disebut **DataFrame**)
- `df.head()` â†’ tampilkan **5 baris pertama** agar kita tahu struktur data

---

### Langkah 2: Periksa Kualitas Data

```python
df.info()
df.isnull().sum()
```

![alt text](images/2.1.2.png)

**Penjelasan:**
- `df.info()` â†’ tunjukkan:
  - Jumlah baris & kolom
  - Tipe data tiap kolom (angka? teks?)
  - Apakah ada data kosong?
- `df.isnull().sum()` â†’ hitung **jumlah nilai kosong** di tiap kolom

---

### Langkah 3: Statistik Ringkas

```python
df.describe().T
```

![alt text](images/2.1.3.png)

**Penjelasan:**
- `df.describe()` â†’ tampilkan **statistik dasar** tiap kolom angka:
  - Rata-rata (mean)
  - Nilai minimum & maksimum
  - Standar deviasi (seberapa tersebar datanya)
- `.T` â†’ â€œtransposeâ€ â†’ ubah baris jadi kolom biar lebih mudah dibaca

---

### Langkah 4: Buang Kolom yang Tidak Perlu

```python

```

![alt text](images/2.1.4.png)

**Penjelasan:**
- Kolom `Id` hanya nomor urut â†’ **tidak membantu analisis**

### Langkah 5: Lihat Distribusi Tiap Fitur

```python
import seaborn as sns
import matplotlib.pyplot as plt

plt.figure(figsize=(15,10))
sns.histplot(data=df, x='SepalLengthCm', kde=True)
# ... (ulangi untuk 3 fitur lain)
plt.show()
```

![alt text](images/2.1.5.png)
![alt text](images/2.1.5.1.png)
**Penjelasan:**
- `histplot` â†’ gambar **histogram**: sebaran nilai suatu fitur
- `kde=True` â†’ tambahkan garis halus yang tunjukkan pola distribusi
- Kita lihat: apakah datanya menyebar merata? Ada puncak? Ada outlier?

> ğŸ” Contoh insight:  
> - `PetalLengthCm` punya **dua puncak** â†’ kemungkinan ada **dua kelompok spesies** yang berbeda!

### Langkah 6: Bandingkan Antar Fitur & Spesies

```python
sns.scatterplot(
    data=df, 
    x='PetalLengthCm', 
    y='PetalWidthCm', 
    hue='Species'
)
```

![alt text](images/2.1.6.png)

**Penjelasan:**
- `scatterplot` â†’ titik-titik yang memetakan dua fitur sekaligus
- `hue='Species'` â†’ warna titik berbeda berdasarkan spesies

> ğŸ¯ Ini adalah **kunci utama EDA Iris!**  
> Hasilnya:  
> - *Setosa* â†’ kelompok terpisah sempurna  
> - *Versicolor* & *Virginica* â†’ tumpang tindih sedikit  
> â¡ï¸ Artinya: **fitur petal sangat kuat untuk membedakan spesies!**

### Langkah 7: Cek Hubungan Antar Fitur (Korelasi)

```python
X = df.drop(columns='Species')
sns.heatmap(X.corr(), annot=True)
```

![alt text](images/2.1.7.png)

**Penjelasan:**
- `X.corr()` â†’ hitung **korelasi** antar fitur (seberapa mirip pola perubahan mereka)
- `heatmap` â†’ visualisasi warna: merah = sangat berkaitan, biru = kurang berkaitan
- `annot=True` â†’ tampilkan angka di tiap sel

> ğŸ”— Temuan:  
> - `PetalLength` & `PetalWidth` â†’ **sangat berkorelasi (0.96)**  
> Artinya: bunga dengan petal panjang biasanya juga lebar.
